# 🇹🇭 Space Hulk: Deathwing - Thai Localization Mod Workflow & Techniques

Repository สำหรับเก็บรวบรวมเทคนิค, เครื่องมือ, และโค้ดสำหรับสร้าง Mod ภาษาไทยให้กับเกม **Space Hulk: Deathwing - Enhanced Edition** (Unreal Engine 4.14.3)

## 📌 เทคนิคที่ค้นพบ (Knowledge Items - KI)

### 1. เทคนิค Font Fallback (ไม่ต้อง Cook ฟอนต์ .uasset)
เกมที่พัฒนาด้วย Unreal Engine 4 มีระบบ Slate UI ซึ่งรองรับระบบ Font Fallback โดยธรรมชาติ หากระบบค้นหาตัวอักษรภาษาไทยในฟอนต์หลักของเกมไม่พบ มันจะวิ่งไปหาฟอนต์สำรองที่ถูกตั้งค่าไว้ที่ `Engine\Content\Slate\Fonts\DroidSansFallback.ttf`

**การนำฟอนต์ไทยเข้าไปในเกมโดยละเอียดที่สุด:**
เทคนิคนี้คือ "ท่าไม้ตาย" ในการแทรกภาษาไทยเข้าไปในเกม UE4 รุ่นเก่า โดยไม่ต้องแตะต้องไฟล์ฟอนต์ดั้งเดิมของเกมเลยแม้แต่ไฟล์เดียว

1. **เตรียมฟอนต์ภาษาไทย:** หาไฟล์ฟอนต์ที่รองรับภาษาไทย (เช่น Tahoma, DB Helvethaica) และนำมาเปลี่ยนชื่อไฟล์ให้เป็นคำว่า **`DroidSansFallback.ttf`** เป๊ะๆ ทุกตัวอักษร
2. **สร้างโครงสร้างโฟลเดอร์จำลอง (Pack):** ภายในโฟลเดอร์สำหรับทำม็อด (ที่เราตั้งชื่อว่า `Pack`) ให้สร้างลำดับชั้นของโฟลเดอร์ย่อยเรียงต่อกันให้ลึกเข้าไปดังนี้:
   ```text
   Pack/
   └── Engine/
       └── Content/
           └── Slate/
               └── Fonts/
                   └── DroidSansFallback.ttf
   ```
3. **วางฟอนต์ให้ถูกที่:** นำไฟล์ `DroidSansFallback.ttf` ที่เตรียมไว้ไปวางไว้ในโฟลเดอร์ `Fonts` ชั้นในสุดให้เรียบร้อย
4. **แพ็ครวมกับไฟล์แปล:** เมื่อจะทำเป็นม็อด ให้นำโฟลเดอร์ `Engine` นี้ไปวางรวมกับโฟลเดอร์ `SpaceHulkGame` (ที่เก็บไฟล์แปล `.locres`) แล้วทำการแพ็คไฟล์เป็นสกุล `.pak` เพียงก้อนเดียว
5. **ความมหัศจรรย์:** เมื่อเข้าเกม ตัวอักษรภาษาอังกฤษจะแสดงฟอนต์สวยงามตามปกติของเกม แต่เมื่อเกมดึงข้อความภาษาไทยขึ้นมาแสดง และหาในฟอนต์ตัวเกมไม่เจอ เอนจิ้นจะดึงฟอนต์ `DroidSansFallback.ttf` ของเราที่ซ่อนไว้มาแสดงผลแทนทันทีแบบอัตโนมัติ!

### 2. การแก้บั๊กใน Pylocres (Integer Hash Bug)
เมื่อทำการนำเข้าไฟล์ `CSV` กลับไปเป็น `.locres` ผ่านคำสั่ง `pylocres from-csv` มักจะเจอข้อผิดพลาด `struct.error: required argument is not an integer` 
- **สาเหตุ:** เกิดจากช่อง `hash` ในไฟล์ CSV ถูกอ่านขึ้นมาเป็นข้อมูลประเภท String (ตัวอักษร) แต่คำสั่งแพ็คเข้ารหัสของโครงสร้าง C++ ต้องการข้อมูลประเภท Integer
- **วิธีแก้:** ต้องทำการแก้ไขโค้ดในตัวไลบรารีของ pylocres โดยตรง (ไฟล์ `cli.py`) ในบรรทัดที่มีการดึงค่า `source_hash` ให้บังคับแปลงเป็น Integer ทันที:
  ```python
  # โค้ดที่ผ่านการแก้บั๊กแล้ว
  source_hash = int(row.get("hash", 0)) if row.get("hash") else 0
  ```

### 3. ระบบแปลภาษาด้วย AI (DeepSeek) & Glossary Injection
ด้วยความที่ซีรีส์ Warhammer 40,000 มีคำศัพท์เฉพาะที่ซับซ้อนมาก เราจึงสร้างสคริปต์ `run_translation_SHDW.py` ที่ทำงานร่วมกับ DeepSeek API โดยมีฟีเจอร์เด่นคือ:
- **Glossary Injection:** มีการฝังพจนานุกรมคำศัพท์เฉพาะเข้าไปใน System Prompt อย่างเข้มงวด (เช่น Space Marine = สเปซมารีน, Heretic = พวกนอกรีต, Genestealer = จีนสตีลเลอร์)
- **Tag Masking:** ป้องกัน AI ทำโค้ดพังโดยการแทนที่โค้ด `<span color="...">` หรือตัวแปรสูตรต่างๆ ด้วย Tag พิเศษ (เช่น `[TAG_0]`) ก่อนส่งให้ AI แปล แล้วค่อยสลับกลับมาคืนเมื่อแปลเสร็จ
- **Checkpoint System:** แปลงไฟล์ทีละแถว และทำการเซฟลง CSV ทันที ป้องกันปัญหาไฟดับ หรือ API ล่มกลางคัน

### 4. เทคนิคบังคับเกมให้อ่านไฟล์ม็อดด้วยสกุล `_P.pak` (Patch File) และคำสั่งบรรจุไฟล์
เกม Unreal Engine รุ่นเก่าๆ (เช่น 4.14) **ยังไม่มีระบบรองรับโฟลเดอร์ `~mods`** ดังนั้นการนำไฟล์ม็อดไปโยนใส่โฟลเดอร์ `Paks` หรือ `~mods` ดื้อๆ เกมจะไม่ยอมโหลดและแสดงภาษาอังกฤษเหมือนเดิม เราต้องบังคับเกมด้วยระบบ Patch

**วิธีแพ็คไฟล์และสร้าง Patch โดยละเอียดที่สุด:**
1. **ใช้เครื่องมือที่ถูกต้อง:** ใช้โปรแกรม `repak.exe` เป็นเครื่องมือหลักในการแพ็คไฟล์
2. **การเขียนคำสั่งแพ็ค:** เปิด Command Prompt หรือสร้างไฟล์ `.bat` โดยใช้คำสั่งนี้ในการแพ็คโฟลเดอร์ `Pack` ของเรา:
   ```cmd
   repak.exe pack Pack SpaceHulkGame-WindowsNoEditor_P.pak --version V3
   ```
   **คำอธิบายเชิงลึก:** 
   - `pack`: สั่งให้โปรแกรมทำงานในโหมดบีบอัด
   - `Pack`: ชื่อของโฟลเดอร์ต้นทางที่เก็บโครงสร้างโฟลเดอร์ม็อด (ที่เราเตรียมโฟลเดอร์ `Engine` และ `SpaceHulkGame` ไว้ข้างใน)
   - `SpaceHulkGame-WindowsNoEditor_P.pak`: **จุดสำคัญ!** เปลี่ยนชื่อไฟล์ปลายทางให้ลงท้ายด้วย `_P` ทันที (`_P` ย่อมาจาก Patch ซึ่งตัวเอนจิ้นจะถูกเขียนโค้ดมาให้โหลดไฟล์ .pak ที่ลงท้ายด้วย `_P` เป็นลำดับสุดท้ายเสมอเพื่อโหลดทับไฟล์เกมหลัก)
   - `--version V3`: **จุดชี้เป็นชี้ตาย!** บังคับให้สร้างโครงสร้างไฟล์ Pak เป็นเวอร์ชัน V3 ซึ่งเป็นเวอร์ชันดั้งเดิมที่ UE 4.14 รองรับ หากไม่ใส่ เกมจะอ่านไฟล์ม็อดนี้ไม่ออกเลย
3. **ติดตั้งม็อด:** นำไฟล์ `_P.pak` ที่ได้ ไปวางไว้คู่กับไฟล์ `.pak` ต้นฉบับของเกม ในโฟลเดอร์ `SpaceHulkGame\Content\Paks\` ของตัวเกม โดยไม่ต้องสร้างโฟลเดอร์ย่อยใดๆ ทั้งสิ้น

## 📂 โครงสร้างของ Workspace

- `/Extracted/` - โฟลเดอร์ที่เก็บไฟล์เกมดั้งเดิมที่ถูกแตกออกมา (ต้นฉบับ)
- `/Pack/` - โฟลเดอร์ที่จำลองโครงสร้างเกม เอาไว้เตรียมนำไปแพ็คเป็น .pak
- `/Translation/` - โฟลเดอร์สำหรับเก็บไฟล์ CSV ที่ Export ออกมา และไฟล์แปลไทย
- `/Tools/` - โฟลเดอร์เก็บโปรแกรมช่วยเหลือ เช่น `repak.exe`
- `1_export_csv.bat` - สคริปต์ดูดข้อความภาษาอังกฤษออกมาเป็น CSV
- `2_import_csv.bat` - สคริปต์ประกอบข้อความภาษาไทยกลับเข้าไปในรูปแบบ .locres
- `3_pack_mod.bat` - สคริปต์แพ็คไฟล์เป็น Mod ท้ายสุด
- `run_translation_SHDW.py` - สคริปต์ AI แปลภาษา

## 🚀 วิธีการใช้งานแบบรวบรัด

1. รัน `1_export_csv.bat` เพื่อดูดข้อความออกมา
2. รัน `run_translation_SHDW.py` เพื่อสั่ง AI แปลภาษาให้เป็นไทย (อย่าลืมลบ API Key ก่อนแชร์!)
3. เมื่อแน่ใจว่าภาษาไทยถูกต้องครบถ้วน ให้รัน `2_import_csv.bat` เพื่ออิมพอร์ต CSV กลับเข้าเป็นไฟล์ Locres
4. รัน `3_pack_mod.bat` เพื่อให้โปรแกรม repak ทำการแพ็คไฟล์ตามคำสั่ง `--version V3` และออกมาเป็นไฟล์ `_P.pak`
5. นำไฟล์ `_P.pak` ที่ได้ไปวางในโฟลเดอร์ Paks ของตัวเกมได้เลย!
